Foire Aux Questions (FAQ)

Retrouvez ici les réponses aux questions les plus fréquentes sur Rumton et les courses colorées.

Comment puis-je m’inscrire à une course ?

Rendez-vous sur la page « Liste des courses », choisissez l’événement qui vous intéresse puis cliquez sur « S’inscrire ». Vous devrez créer un compte ou vous connecter pour finaliser votre inscription.

Comment devenir organisateur ?

Lors de l'inscription vous pouvez sélectionner le rôle organisateur et indiquer les raisons de votre demande. La demande pour devenir organisteur est aussi disponible depuis votre profile si vous êtes participant. A la suite de votre demande, un administrateur devra approuver votre demande.

Comment créer ma propre course colorée ?

Après connexion, cliquez sur « Créer une course » dans le menu. Remplissez le formulaire avec les informations de votre événement, choisissez une cause à soutenir et validez. Votre course sera alors visible par toute la communauté. Attenttion pour créer un course vous devez être organisateur

Les courses sont-elles payantes ?

Oui, chaque course est payante. Le tarif est fixé par l’organisateur et une partie des fonds est reversée à la cause associée à l’événement.

Comment obtenir mon dossard ?

Une fois inscrit à une course, vous pouvez télécharger votre dossard depuis la page de l’événement ou depuis votre espace personnel.

Puis-je participer si je ne suis pas sportif ?

Bien sûr ! Les courses colorées sont ouvertes à tous, quel que soit votre niveau. L’objectif est de s’amuser et de partager un moment convivial.

Comment soutenir une cause ?

Lors de l’inscription à une course, vous soutenez automatiquement la cause associée à l’événement. Vous pouvez aussi découvrir toutes les causes sur la page dédiée.

Comment contacter l’équipe Rumton ?

Pour toute question ou suggestion, écrivez-nous à contact@rumton.fr.
